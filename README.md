# Rick and Morty Web App

This is a simple React web application that displays information about characters from the TV show Rick and Morty.

## Getting Started

To get a local copy of the project up and running, follow these steps:

### Prerequisites

Make sure you have Node.js and npm installed on your local machine.

### Installation

1. Clone the repository to your local machine:

   bash
   git clone https://github.com/your-username/rick-and-morty-web-app.git

2. Navigate into the project directory:

   bash
   cd rick-and-morty-web-app
   

3. Install the project dependencies:

   bash
   npm install
   

### Usage

To start the development server and view the project in your browser, run:

bash
npm start


The project will be running at [http://localhost:3000](http://localhost:3000).
